# Roman Numerals Workshop

This is a workshop to practice test driven development and refactoring in Java. The goal is to implement a converter
from Roman numerals to Arabic numerals and vice versa.

## Getting Started

Run:

```
mvn clean install
```

## Arabic to Roman
